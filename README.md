怎样挑选水果（安卓版）
======================

How to pick and choose fruits for Android. (Pick And Choose, PAC)

顺手写的一个很简单的安卓小程序，为您挑选水果提供有限指导。

  - 现在是夏天，怎样挑又大又甜又脆的大西瓜呢？
